Actividad 5 - Arquitectura de Persistencia SmartRoute / FlashRoute DSS
Este repositorio contiene el diseño de persistencia del sistema DSS para FlashLogistics. Incluye el diagrama de clases UML, el modelo relacional y el script SQL inicial con tablas, llaves primarias, llaves foráneas, restricciones CHECK, UNIQUE, NOT NULL, DEFAULT, índices y vistas analíticas para soportar decisiones sobre rutas, pedidos, combustible, alertas y KPIs.
datos correspondientes a la actividad5 carpeta models y sql.
